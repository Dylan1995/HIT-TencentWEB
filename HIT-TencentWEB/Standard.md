一、相关页面的名称
主页：index.html
新闻页：news.html
个人登陆页：member.html
成员风采展示页：membershow.html
俱乐部资源页：source.html

二、具体开发注意事项
1、css，js代码与html代码最好分开
2、采用mvc模式开发，前端代码放入view中，后台放入controller中，数据放入model中（包括图片，成员信息，资源目录等）
3、遇到问题及时在群里面提问，共同商讨
4、每天开发前记住先在命令行输入 “git pull” 同步，再进行开发。
5、提交代码记住写上commit -m “修改的内容”

三、开发语言
前端：html css javascript bootstrap.js框架 
后台：php
数据库：mysql

（建议大家自己的本地电脑安装wamp集成虚拟服务器）
